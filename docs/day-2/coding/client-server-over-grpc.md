# Client Server using gRPC and UNIX Domain Sockets

## Preparation

### Mariner

```
sudo tdnf install -y \
    protobuf \
    grpc \
    grpc-plugins \
    grpc-devel \
    c-ares-devel \
    re2-devel
```

### Ubuntu

```
sudo apt install -y \
    protobuf-compiler-grpc \
    libprotobuf-dev \
    libgrpc-dev
```
